## Reddit (r/webdev, r/reactjs)

**Title: I built an Open Source Farm-to-Table Marketplace with Route Optimization (MERN Stack)**

Hey everyone,

I just open-sourced **AgriDirect**, a platform I've been working on to help farmers sell directly to consumers.

**Why I built it:**
To solve the "last mile" delivery problem for small-scale farmers and eliminate middlemen.

**Tech Stack:**
*   **Frontend**: React, Vite, Leaflet Maps
*   **Backend**: Node.js, Express, MongoDB
*   **Real-time**: Socket.io
*   **DevOps**: Docker, GitHub Actions

**Coolest technical challenge:**
Implementing the route optimization system. I used the Haversine formula and a Nearest Neighbor algorithm to sort delivery stops automatically for farmers.
